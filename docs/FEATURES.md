# Admin web — features overview

Next.js admin UI under `frontend/src/app/`. Capability list can drift — verify against live routes.

## Admin web (`frontend/`)

Login; dashboard; users & villas; resident management; maintenance & billing; bank accounts; complaints & analytics; vendors; notices; gates; guard shifts & patrols; visitors & pre-approved visitors; parcels; amenities & bookings & calendar; staff / vehicles / parking; polls; documents; incidents; banners; expenses & categories & summaries; SOS / water / garbage / notifications pages as present under `src/app/`.

Super admin: `/super-admin` and `/super-admin/login` (platform operator, separate from society admin).
